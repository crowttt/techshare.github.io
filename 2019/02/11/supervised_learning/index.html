Supervised learning （監督學習）

**

引言

思考一個實際的例子，假設我們有一筆資料，紀錄一個地區的房屋的面積和房價。那麼有沒有辦法用這些資料得到面積和房價之間的關係，也就是說，給定一棟房子的面積，可以預測他的價格。

**

符號定義


**

  • x(i) : 一個輸入變數 (input variable)

  • y(i) : 一個輸出變數 (output variable)

  • (x(i), y(i)) : 一組訓練樣本 (training example)

  • {x(i), y(i)} , i = {1,2,3…..m} : 訓練樣本集 (training set)

  • h (hypothesis) : 經過訓練得來的函式

    • **

    訓練流程

    **
    參考上面的流程圖

    Training set 就是訓練樣本集，也就是上面表格列的資訊，將表格裡的資訊輸入 Learning algorithm 後，我們的演算法可以訓練出一組參數，通常用 θ 表示。
    參數 θ 與變數 x 的線性組合就是 hypothesis，一個表達面積與房價之間關係的方程式。

    得到 hypothesis 後帶入 x 也就是房屋面積，便可以估計出 y 房價。
    **

    問題種類：

  • 回歸問題（regressionproblem）:
    • 如果目標變數（y）是連續的，像是範例的房價問題，我們稱這個問題是一個回歸問題。
  • 分類問題（classification problem）:
    • 如果目標變數是離散的，（例如給定居區域，求是獨棟還是公寓）則稱這個問題是一個分類問題。
